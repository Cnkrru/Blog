# 博客管理工具使用文档

## 工具简介

博客管理工具是一个基于PyQt5开发的GUI应用程序，用于管理Vue博客项目。它提供了丰富的功能，包括文章管理、站点维护、分类管理、标签管理、项目管理、工具管理、链接管理、媒体管理、SEO优化和性能监控等。

## 安装说明

### 系统要求
- Python 3.7+
- PyQt5
- 其他依赖包

### 安装步骤
1. 确保已安装Python 3.7或更高版本
2. 进入工具目录：`cd py-blog-tool`
3. 创建虚拟环境：`python -m venv venv`
4. 激活虚拟环境：
   - Windows: `venv\Scripts\activate`
   - Linux/Mac: `source venv/bin/activate`
5. 安装依赖：`pip install -r requirements.txt`
6. 运行工具：`python ui_tool/main.py` 或双击 `run.bat`

## 使用指南

### 基本操作
1. **启动工具**：双击 `run.bat` 或运行 `python ui_tool/main.py`
2. **选择功能**：在左侧导航栏选择需要的功能模块
3. **操作数据**：根据需要进行添加、编辑、删除等操作
4. **保存更改**：所有修改都会自动保存到相应的配置文件

### 高级操作
1. **批量操作**：在文章管理模块中，选择多个文章后点击批量操作按钮
2. **自动备份**：在站点维护模块中，设置自动备份选项
3. **性能监控**：在性能监控模块中，点击开始监控按钮分析站点性能
4. **SEO优化**：在SEO优化模块中，生成站点地图和RSS订阅源

## 常见问题

### 1. 工具无法启动
- 检查Python是否正确安装
- 检查虚拟环境是否激活
- 检查依赖包是否安装完整

### 2. 无法加载文章/分类/标签
- 检查博客项目路径是否正确
- 检查相关配置文件是否存在
- 检查文件权限是否正确

### 3. 上传文件失败
- 检查媒体目录是否存在
- 检查文件权限是否正确
- 检查文件大小是否超限

### 4. 性能监控失败
- 检查本地开发服务器是否启动
- 检查网站URL是否正确
- 检查网络连接是否正常

## 注意事项

1. **数据备份**：在进行重要操作前，建议先备份博客数据
2. **文件权限**：确保工具有足够的权限读写博客文件
3. **路径配置**：确保博客项目路径正确配置
4. **版本兼容性**：工具适用于Vue 3博客项目
5. **性能优化**：对于大型博客，部分操作可能需要较长时间
